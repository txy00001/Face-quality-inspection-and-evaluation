## 二、训练
  选择训练集和验证集的txt即可训练，两种方案均已给出，执行相应训练文件即可


## 三、转onnx
  运行pytorch2onnx.py ,选择要转的pth文件和导出的文件名即可

## 四、推理
  选择需要测试的模型，模型保存在checkpoins文件夹中,运行demo.py,程序会自动验证测试集前50个案例。
